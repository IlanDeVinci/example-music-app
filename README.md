# example-music-app

Laravel Vue.js example music application with user authentication, track management, playlists, and API key access.

## How to run:

1. Clone the repository:
    ```
    git clone
    ```
2. Install dependencies:
    ```
    composer install
    npm install
    ```
3. Set up your `.env` file
4. Run migrations:
    ```
    php artisan migrate (--seed)
    ```
5. Start the development server:
    ```
    php artisan serve
    npm run dev
    ```
6. Access the application at `http://localhost:8000`.
